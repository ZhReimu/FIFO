# 嵌入式大作业

基于Linux的FIFO管道通信机制的 仿真与设计

# 文件说明

## cs.h

公共头文件，`生产者` 和 `消费者` 都需要用到的一些 `include` 与函数声明

## cs.c

公共模块，`cs.h` 的实现部分

## write.c

`生产者` 具体实现

## read.c

`消费者` 具体实现

## Makefile

用 `make` 编译 `生产者` 和 `消费者` 用的  `Makefile`

## msys-2.0.dll

因为我是使用 `Mysys2` 在 `Windows` 下编译的，想要离开 `Mysys2` 的终端就必须带上这个 `Posix` 兼容库

# 使用方法

使用 `make` 命令编译 `生产者` 与 `消费者`

先开启 `生产者` 程序，再开启 `消费者` 程序
